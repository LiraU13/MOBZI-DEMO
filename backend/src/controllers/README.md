# Controladores

Esta carpeta contiene los controladores que manejan la lógica de negocio.

## Estructura sugerida:

- `auth.controller.ts` - Controlador de autenticación
- `usuario.controller.ts` - Controlador de usuarios
- `ruta.controller.ts` - Controlador de rutas
- `admin.controller.ts` - Controlador de administración
- etc.

Cada controlador maneja las requests HTTP y llama a los modelos correspondientes.
