# Utilidades

Esta carpeta contiene funciones de utilidad reutilizables.

## Estructura sugerida:

- `validation.ts` - Funciones de validación
- `encryption.ts` - Funciones de encriptación/hashing
- `formatting.ts` - Funciones de formateo de datos
- `errors.ts` - Clases de error personalizadas
- `logger.ts` - Utilidades de logging
