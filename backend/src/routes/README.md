# Rutas

Esta carpeta contiene las definiciones de rutas de la API.

## Estructura sugerida:

- `auth.routes.ts` - Rutas de autenticación
- `usuario.routes.ts` - Rutas de usuarios
- `ruta.routes.ts` - Rutas de rutas
- `admin.routes.ts` - Rutas de administración
- `index.ts` - Agrupa todas las rutas

Cada archivo de rutas define los endpoints y conecta con los controladores.
